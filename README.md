# SkinTrack

Mobile app for skin and wellness tracking (React Native, iOS + Android).

## Repository Structure

```text
ForSki/
├── src/                 # app features, screens, services
├── ios/                 # iOS project
├── android/             # Android project
├── docs/                # product and sale docs
├── package.json
└── README.md
```

## Product Docs

- `docs/PRODUCT_OVERVIEW.md`
- `docs/ROADMAP.md`
- `docs/SALE_CHECKLIST.md`

## Quick Start

```bash
npm install
npm run lint
npx tsc --noEmit
```

Start Metro:

```bash
npm start
```

Run iOS (new terminal):

```bash
npx react-native run-ios
```

Run Android (new terminal):

```bash
npx react-native run-android
```
