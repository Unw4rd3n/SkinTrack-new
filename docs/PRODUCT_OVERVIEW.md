## What It Is

SkinTrack is a mobile app for daily skin tracking with a focus on practical actions:

- daily check-in (skin + wellness),
- cycle calendar,
- care planner (retinol/acids safety logic),
- reminders,
- diary and notes by date,
- analytics and trends.

## Tech Stack

- React Native (iOS + Android)
- TypeScript
- React Navigation
- Redux Toolkit
- WatermelonDB

## Current Positioning

- Wellness / beauty tracking app
- Not a medical device
- Action-first UX for daily use
